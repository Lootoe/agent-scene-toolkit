# Linear设计风格指南

## 核心思想

Linear设计风格是一整套完整的设计哲学和美学体系：

1. 专注与高效：界面设计旨在减少视觉噪音和干扰，让用户专注于核心任务。 操作流程被设计为线性和直观的，避免在单一视图中提供过多选择。
2. 专业与精准：整体质感追求像素级完美，通过精细的细节（如1像素边框、精确对齐）来体现其专业性。 
3. 克制与优雅：动效和视觉元素的使用非常克制，只在必要时用于提供反馈或引导注意力，避免任何不必要的装饰和夸张效果。

## Linear设计风格的具体规范

### 色彩

Linear的色彩策略以简约和高对比度为核心，营造专注、无干扰的视觉体验。

- 基础色板：主要由中性色构成，为内容和交互元素提供一个稳固的背景。

- - 深色背景：通常使用近乎纯黑的深灰色，而非纯黑（#000000），以避免过高的对比度导致视觉疲劳。
  - 中性灰：用于界面元素的层次区分，如卡片、侧边栏和分割线。
  - 纯白：用于高亮文本和关键图标，确保在深色背景下具有最佳可读性。

- 强调色

- - 主强调色：需要一个主强调色用于可交互元素，如按钮、链接和选定状态，起到引导用户视觉焦点的作用。
  - 辅助强调色：可以使用小范围的、饱和度较低的冷色调（如冷蓝色、墨绿色）作为辅助或状态色（如成功、警告）。

- 渐变：渐变是Linear风格的灵魂之一，通常用于背景装饰、图标或高光效果。

- - 应用方式：常使用线性渐变（linear-gradient()）和径向渐变（radial-gradient()）。
  - 色彩组合：可以是品牌主色或高饱和度的对比色组合，用于创造动态、神秘的视觉效果。

### 字体与排版

排版遵循清晰、统一和结构化的原则，确保信息层次分明且易于阅读。

- 字号与层级：使用模块化比例（如1.25的比例因子）来定义标题、副标题、正文和标签等不同角色的字号。


- 行高：

- - 通用法则：对于正文等较小字号的文本，行高通常设置为字号的1.5倍，以保证最佳的阅读舒适度。
  - 标题行高：对于较大的标题，行高可以适当减小，例如1.2倍。

- 间距：

- - 8点网格系统：强烈推荐使用8点网格系统来规范所有元素的间距、边距和内边距（padding），即所有尺寸都应是8的倍数（如8px, 16px, 24px, 32px）。这能确保布局的协调性、节奏感和一致性。

### 界面元素与效果

Linear风格的精致感体现在对每个界面细节的精雕细琢上。

- 边框：

- - 极细描边：使用0.5px到1px的极细边框来勾勒组件轮廓，创造出一种锋利、精确的感觉。 颜色通常是比背景稍亮的灰色。

- 模糊与噪点：

- - 背景模糊：常用于模态框（Modal）或弹出菜单的背景，通过模糊底层内容来增强前景元素的焦点。这可以通过CSS的backdrop-filter: blur()来实现。
  - 微噪点：在深色背景或渐变上覆盖一层非常细微的噪点纹理，可以增加界面的深度和质感，避免纯色背景的单调。

- 动效：

- - 原则：动效设计必须服务于功能，保持流畅、克制且响应迅速。 动画时长通常很短，旨在提供即时反馈。
  - 应用场景：常用于按钮悬停、菜单展开、状态切换等交互时刻。
